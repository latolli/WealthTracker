# WealthTracker
**Tool for tracking monthly net wealth progression**

- Fun little project to learn PyQt and test GitHub Copilot
- Goal of the tool is to motivate user to spend less on useless junk and invest & save more

- Tool allows you to track net wealth progression, monthly income and expenses, and different types of expenses you have

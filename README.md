# WealthTracker

**A simple desktop tool for tracking your monthly net wealth progression**

WealthTracker is a small personal project built to explore PyQt and experiment with GitHub Copilot. The main goal is to help users become more mindful of their finances—spend less on unnecessary purchases and focus more on saving and investing.

---

## ✨ Features

### 📊 Track your finances
- Monitor **net wealth progression over time**
- Log **monthly income and expenses**
- Categorize and analyze **different types of expenses**
